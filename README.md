# IBM System/23 Datamaster Miniprobe

This is not my circuit design... but it is my PCB layout. I used SOIC and 0805 devices to make this small.

The 14 pin connector fits over a set of 8 and 4 pins on the main logic board that has 2 pins of empty spacing between them. I recommend you shove some hot glue or something similar into the 2 unused pins on the connector to help you align the PCB onto the pins sticking up from the main board.
